== Description ==

Last.fm for WordPress displays your recently listened tracks in your WordPress blog.

**Features**

    * Simply
    * Customizable
    * Widget support
    * No options page (yes, it is a feature)
    * Uses Wordpress resources (no extra files needed)

**Usage**

If you use WordPress widgets, just drag the widget into your sidebar and configure. If widgets are not your thing, use the following code to display your recently listened tracks:

`<?php lastfm_tracks("username"); ?>`

For more info (options, customization, etc.) visit [the plugin homepage](http://rick.jinlabs.com/code/lastfm "Last.fm for Wordpress").
